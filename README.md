# Translatable

## Requirements

* ActiveRecord 3.2
* I18n

## Installation

To install just use:

```ruby
gem install easy_translatable
```

When using bundler put this in your Gemfile:

```ruby
gem 'easy_translatable', '~> 0.1.0'
```
